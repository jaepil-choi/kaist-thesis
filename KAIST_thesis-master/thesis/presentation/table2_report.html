TNIC vs. 오토인코더 동종업체 분류 — 파마-맥베스 회귀 결과

저자

KAIST 금융공학대학원

공개

2026년 3월 26일

개요

본 보고서는 Hoberg and Phillips (2018)의 Table 2를 한국 주식시장 데이터로 복제한 수익률 공변동 분석 결과를 제시한다. 동종업체 집단을 정의하는 두 가지 방법론을 비교한다.

  • TNIC (텍스트 기반 네트워크 산업 분류): H&P (2016) 방법론에 따라 한국 DART 공시 사업보고서 텍스트 간의 코사인 유사도를 이용하여 동종업체를 정의한다. 동종업체 수익률은 동일가중(EW) 방식으로 산출한다.
  • 오토인코더 (Kim et al. 2020): 동일한 한국어 텍스트 데이터에 심층 오토인코더(2000→500→125→10→125→500→2000)를 적용하여 10차원 잠재 임베딩을 학습하고, 임베딩 간 코사인 유사도(임계값 0.20)로 동종업체를 정의한다. 동종업체 수익률은 동일가중(EW) 방식으로 산출한다.

두 방법 모두에서, 비교 기준이 되는 전통적 분류는 FnGuide 산업 분류(SIC 코드의 한국 대응 분류)이며, 이를 기반으로 시가가중(VW) 동종업체 수익률을 산출한다.

분석의 핵심 질문은 다음과 같다: TNIC 또는 오토인코더 기반 동종업체 수익률이 FnGuide 동종업체 수익률이 이미 설명하는 부분을 넘어서, 개별 기업의 자기 수익률을 설명하거나 예측하는 데 추가적인 정보를 제공하는가?

전처리 개선 사항 (이번 결과 반영): 오토인코더 결과는 다음의 전처리 수정이 적용된 새로운 한국어 코퍼스 기반이다. (1) 의존명사(NNB 품사) 제거 — 일반명사(NNG)·고유명사(NNP)만 추출; (2) 지리적 용어 필터 확장(~80개 도시·국가·광역 지명); (3) 빈도 필터 적용 후 20-단어 최소 기준 재검증. 이 수정들은 전통적 TNIC 파이프라인의 한국어 텍스트 전처리 품질을 개선하여, 이전에 보고된 결과와 계수 크기가 달라질 수 있다.

  • 패널 A: 종속변수 = \(t\)월 자기 수익률 (동시적 공변동)
  • 패널 B: 종속변수 = \(t+1\)월 자기 수익률 (예측 회귀)

횡단면 표준화 (Z-점수 변환)

파마-맥베스 회귀를 실행하기 전에, 모든 독립변수(통제변수 및 동종업체 수익률 변수 포함)를 매월 횡단면 내에서 평균 0, 표준편차 1이 되도록 표준화한다.

\[z_{i,t}(X) = \frac{X_{i,t} - \bar{X}_t}{\sigma_{X,t}}\]

여기서 \(\bar{X}_t\)와 \(\sigma_{X,t}\)는 \(t\)월 변수 \(X\)의 횡단면 평균과 표준편차이다. 이 표준화는 두 가지 역할을 한다.

  1. 계수 비교 가능성: 추정된 기울기 계수는 동종업체 수익률의 1표준편차 충격에 대한 자기 수익률의 반응을 나타내므로, TNIC와 SIC 계수를 원시 수익률 분포의 차이와 무관하게 크기 면에서 직접 비교할 수 있다.
  2. 수치 안정성: 횡단면적 규모 이질성을 제거하여 각 기간 횡단면 OLS의 조건수(condition number)를 개선한다.

수익률 계산 과정에서 발생하는 무한대(Inf) 값은 표준화 전에 결측치(NaN)로 처리된다. 표본 전체에 걸쳐 유효 관측치가 전혀 없는 종목(all-NaN)은 표준화 전에 제외된다.

파마-맥베스 추정 절차

파마-맥베스(1973) 추정량은 다음 절차로 구현된다.

  1. 월별 횡단면 OLS: 표준화된 변수를 사용하여 매월 \(t\)마다 횡단면 회귀를 실행한다. \[r_{i,t} = \alpha_t + \boldsymbol{\beta}_t' \mathbf{X}_{i,t} + \varepsilon_{i,t}\] 해당 설정의 모든 회귀변수에 대해 완전 관측치(listwise deletion)를 사용한다.

  2. 시계열 평균: 월별 계수 벡터의 시계열 평균을 구한다. \[\hat{\boldsymbol{\beta}} = \frac{1}{T} \sum_{t=1}^{T} \hat{\boldsymbol{\beta}}_t\]

  3. 표준오차: H&P (2018) Table 2의 관행에 따라 뉴이-웨스트(Newey-West, 1987) 2시차 수정을 적용하여 월별 추정치 시계열의 자기상관을 보정한다.

동시 공변동 비교 (패널 A)

두 방법 모두 \(t\)월에서 크고 고도로 유의미한 동시 동종업체 수익률 계수를 도출한다.

지표 TNIC (Spec 1) 오토인코더 (Spec 1) SIC 단독 (Spec 3)
동종업체 계수 (\(t_0\)) 0.0127*** 0.0053*** 0.0149***
\(t\)-통계량 17.04 7.08 15.74
SIC 계수 (\(t_0\), 병렬 통제) 0.0141*** 0.0134*** —

주요 관찰:

  • TNIC 통제 후 SIC 계수 감소: Spec 3의 0.0149에서 Spec 1의 0.0141로 약 5.4% 감소. TNIC가 FnGuide 분류에 포함되지 않는 추가 정보를 반영함을 시사한다.
  • 오토인코더 통제 후 SIC 계수 감소폭 유사: 0.0149에서 0.0134로 약 10.1% 감소. 개선된 전처리 적용 후, 오토인코더 동종업체와 FnGuide 그룹핑의 중첩이 이전 결과(27.5%)보다 줄어들었다. 이는 지리 용어 및 의존명사 필터 수정으로 코퍼스가 산업 특이적 어휘에 더 집중하게 됐기 때문으로 해석된다.
  • Spec 5의 AE\(_t\) 부호 역전 (−0.0080***): \(t\)~\(t{-6}\) 시차가 모두 포함된 Spec 5에서 AE\(_t\) 계수가 음수로 역전된다. 이는 다중공선성에 기인한 억제 효과(suppression effect)이며, 동시적 계수 대신 Spec 1, 2 결과를 기준으로 해석해야 한다.
  • 시차 공변동 (Spec 6, 7): TNIC의 \(t{-1}\)~\(t{-3}\) 시차 계수는 Spec 7의 TNIC\(_{t-1}\) (0.0010*, \(t\) = 2.31)만 경계선상에서 유의하다. 오토인코더의 경우, 개선된 전처리 적용 후 AE\(_{t-1}\) 계수가 모든 설정에서 통계적으로 유의하지 않다 (Spec 6: \(t\) = 1.73; Spec 7: \(t\) = 0.51). 이는 이전 결과(AE\(_{t-1}\)이 Spec 6에서 0.0021***, Spec 7에서 0.0024***)와 비교하여 주목할 만한 변화이다.

수익률 예측력 비교 (패널 B)

예측 회귀(패널 B, 종속변수 \(t+1\))는 동시 정보를 배제한 보다 엄격한 검증이다.

지표 TNIC (Spec 1) 오토인코더 (Spec 1) SIC 단독 (Spec 3)
동종업체 계수 (\(t_0\) → \(r_{t+1}\)) 0.0007 (1.64) 0.0012 (1.60) 0.0032*** (3.66)
SIC 계수 (\(t_0\) → \(r_{t+1}\), 병렬) 0.0036*** 0.0029*** —

결과는 다음 순위를 보여준다.

  1. TNIC 동종업체 수익률 (\(t\))의 \(t+1\) 예측력은 약하다 (Spec 1: 계수 0.0007, \(t\) = 1.64, 통계적으로 유의하지 않음; Spec 2와 Spec 4~5에서만 한계적으로 유의).
  2. 오토인코더 동종업체 수익률 (\(t\))의 \(t+1\) 예측력도 Spec 1에서 유의하지 않다 (계수 0.0012, \(t\) = 1.60). SIC 통제 없이 추정한 Spec 2에서만 한계적으로 유의하다 (0.0021*, \(t\) = 2.55). 이는 전처리 개선 이전 결과(Spec 1: 0.0020**, \(t\) = 3.12)와 비교하여 예측력이 약화된 것을 보여준다.
  3. SIC 동종업체 수익률 (\(t\))의 단독 예측력이 가장 강하다 (Spec 3: 0.0032***, \(t\) = 3.66).

오토인코더 예측력 약화의 가능한 해석:

  • 더 순수한 산업 특이적 어휘: 개선된 전처리(의존명사·지리 용어 제거)는 코퍼스를 사업 내용에 특화된 용어로 정제하여, 이전 코퍼스가 부분적으로 포착하던 지역·조직 수준의 공통 정보를 제거했을 수 있다.
  • FnGuide와의 중첩 감소: SIC 계수 감소폭이 27.5% → 10.1%로 줄어든 것은 오토인코더 신호가 FnGuide와 덜 겹치게 됐음을 의미한다. 이 경우 SIC 통제 후 오토인코더의 설명력이 줄어들 수 있다.

SIC 단독 계수(Spec 3, 패널 B: 0.0032***)는 두 비교 표본에서 동일하게 유지되어(\(N\) = 315,062), 깨끗한 벤치마크 역할을 한다.

두 동종업체 집단 방법 모두 한국 주식시장에서 경제적으로 유의미한 동시 수익률 공변동을 보여주지만, 개선된 전처리 적용 후 오토인코더의 차월 수익률 예측력은 통계적으로 유의하지 않은 수준으로 약화됐다. TNIC와 오토인코더 모두 SIC를 통제한 상태에서 \(t+1\) 예측력이 없다는 점에서, 두 방법이 포착하는 정보의 상당 부분이 FnGuide 분류와 중첩되거나, 한국 시장의 정보 전달 속도가 빠름을 시사한다. TNIC는 투명한 임계값 기반의 이진 동종업체 정의를 제공하며, 오토인코더는 더 넓은 커버리지와 밀집된 동종업체 집합을 제공하는 상호 보완적 방법론이다. 두 방법론의 비교에서 가장 뚜렷한 차이는 오토인코더의 동시 계수 크기가 이전 버전 대비 현저히 감소한 점으로, 이는 전처리 품질이 추정 결과에 실질적인 영향을 미침을 보여준다.
