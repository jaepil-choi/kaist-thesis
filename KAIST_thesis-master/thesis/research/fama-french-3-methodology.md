기업규모(Sizei=Pi X Ni): t년 6월말 보통주의 시가총액 사용

장부가치 대 시장가치 비율(B/Mi=Bi/(PiXNi))는 자기자본의 장부가치를시장가치로 나눈다.

자기자본의 장부가치: t-1년 12월말의 보통주 자본금에 자본잉여금,이익잉여금, 자기주식, 이연법인세 부채를 더해 측정

시장가치: 12월 말의 보통주 주가에 발행주식을 곱해 측정한다.

---

파마-프랜치 3 팩터(SMB, HML)는 **포트폴리오 이중 정렬(portfolio sorting)** 방법을 사용하여 구성합니다.

핵심은 **매년 6월 말(year *t*)**을 기준으로 포트폴리오를 재구성(rebalancing)하는 것입니다. 6월을 기준으로 삼는 이유는 **선행 편향(look-ahead bias)을 피하기 위함**입니다. 즉, 전년도(year *t-1*) 12월 말 기준의 재무제표(BE)가 시장에 확실하게 공표되었을 시점이기 때문입니다.

---

## 팩터 생성 방법 (2x3 이중 정렬)

1.  **정렬 기준 (매년 6월 말, *t* 시점):**
    * **규모(Size):** 모든 주식을 **`ME` (시가총액)** 기준으로 정렬하여 상위 50% (Big)와 하위 50% (Small)로 나눕니다.
    * **가치(Value):** 모든 주식을 **`BE/ME` (장부가-시가 비율)** 기준으로 정렬하여 상위 30% (High), 중위 40% (Medium), 하위 30% (Low)로 나눕니다.

2.  **6개 포트폴리오 구성:**
    위 두 기준을 독립적으로 교차하여 6개의 포트폴리오를 만듭니다. (가중평균 수익률 사용)
    * **S/L:** Small Size / Low BE/ME
    * **S/M:** Small Size / Medium BE/ME
    * **S/H:** Small Size / High BE/ME
    * **B/L:** Big Size / Low BE/ME
    * **B/M:** Big Size / Medium BE/ME
    * **B/H:** Big Size / High BE/ME

3.  **팩터 수익률 계산:**
    이 6개 포트폴리오의 **다음 12개월 (year *t*년 7월 ~ year *t+1*년 6월)**간의 월별 수익률을 이용해 팩터를 계산합니다.

    * **SMB (Small Minus Big):** (소형주 수익률) - (대형주 수익률)
        $$
        SMB = \frac{1}{3}(R_{S/L} + R_{S/M} + R_{S/H}) - \frac{1}{3}(R_{B/L} + R_{B/M} + R_{B/H})
        $$

    * **HML (High Minus Low):** (가치주 수익률) - (성장주 수익률)
        $$
        HML = \frac{1}{2}(R_{S/H} + R_{B/H}) - \frac{1}{2}(R_{S/L} + R_{B/L})
        $$

---

## BE/ME 계산 시점 (핵심)

가장 헷갈리기 쉬운 부분이며, 질문하신 핵심입니다. **규모(Size) 정렬에 쓰는 ME**와 **가치(BE/ME) 정렬에 쓰는 ME**의 기준 시점이 다릅니다.

포트폴리오 구성 시점(rebalancing date)을 **year *t*년 6월 말**이라고 할 때:

| 변수 (Variable) | 기준 시점 (Timing) | 설명 (Description) |
| :--- | :--- | :--- |
| **규모 (Size) 정렬** | **Year *t*년 6월 말** ME | 포트폴리오 구성 시점의 시가총액 |
| **가치 (Value) 정렬 (BE/ME)** | | |
| &nbsp;&nbsp;&nbsp; -> **BE (장부가치)** | **Year *t-1*년 12월 말** | 전년도 말 재무제표 상의 장부가치 |
| &nbsp;&nbsp;&nbsp; -> **ME (시가총액)** | **Year *t-1*년 12월 말** | **전년도 말** 기준 시가총액 |

**요약:**

* **Size 정렬용 ME:** **현재 (year *t*년 6월 말)**의 시가총액을 사용합니다.
* **BE/ME 비율 계산용 BE:** **6개월 전 (year *t-1*년 12월 말)** 재무제표 값을 사용합니다.
* **BE/ME 비율 계산용 ME:** **6개월 전 (year *t-1*년 12월 말)** 시가총액을 사용합니다.

즉, BE/ME 비율 자체는 6개월 전 데이터(t-1년 12월 말 기준)를 사용하여 계산하고, 이 계산된 비율을 가지고 6월 말(t년 6월 말)에 주식들을 정렬합니다. 반면 '규모' 정렬은 순수하게 6월 말(t년 6월 말)의 시가총액을 기준으로 합니다.
